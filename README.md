# Particle Trajectories in General Relativity
## Coded by Emily Troutman and Alex Huff

This repo contains our final project for PHYS 3120: Computational Physics. The Simulation.ipynb is the actual simulation. As a default, it was input parameters set to simulate the Schwarzchild metric in areal coordinates. This can easily be changed. The outputs folder contains different interesting outputs to analyze using our graphics viewer.
